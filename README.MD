# kipon-ui
This is the git repository for a simple angular 7 ui library.

## Installation

To install this library, run:

```bash
$ npm install kipon-ui --save
```

### Documentation
Follow the link to the npm package for a full documentation

[goto npm package view](https://www.npmjs.com/package/kipon-ui)
